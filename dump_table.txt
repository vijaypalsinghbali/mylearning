For dump with data:

For no data only schema:
mysqldump -h <hostname> --set-gtid-purged=OFF -u <user> -p --no-data db_name tablename > dump_tablename.sql


For restore without data only schema:
mysql -h <hostname> -u root -p db_name < dump_tablename.sql



If restore is not happening because of super privileges use below steps:
Log in to RDS
Go to the required database 
Run command ; source /root/dump_db_name.sql where dump_db_name.sql iss dump file available on the same server
